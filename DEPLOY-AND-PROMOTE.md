# 商业银行AI工具包 - 假门落地页部署 & 短视频引流指南

## 一、部署落地页到公网（2种方式任选）

### 方式A：GitHub + Vercel（推荐，后续更新方便）

1. **安装Git**
   - 打开 https://git-scm.com/download/win 下载安装
   - 安装时一路默认，只需注意勾选"Add to PATH"

2. **初始化仓库并推送**
   ```bash
   cd C:\Users\45846\WorkBuddy\2026-05-25-15-44-27\bank-ai-landing
   git init
   git add .
   git commit -m "feat: 假门落地页v1"
   
   # 在GitHub创建仓库 bank-ai-landing（公开）
   git remote add origin https://github.com/你的用户名/bank-ai-landing.git
   git branch -M main
   git push -u origin main
   ```

3. **Vercel部署**
   - 打开 https://vercel.com 用GitHub登录
   - 点击"Add New Project" → 选择bank-ai-landing仓库
   - Framework Preset: Vite
   - Build Command: `npm run build`
   - Output Directory: `dist`
   - 点击Deploy，等待1分钟
   - 获得公网URL，如 `https://bank-ai-landing.vercel.app`

4. **后续更新**
   - 改代码 → `git add .` → `git commit -m "update"` → `git push`
   - Vercel自动重新部署，1分钟生效

### 方式B：Netlify Drop（无需Git，1分钟上线）

1. 打开 https://app.netlify.com/drop
2. 把 `bank-ai-landing/dist` 文件夹直接拖进去
3. 等30秒，获得公网URL
4. 缺点：后续更新需要重新拖拽上传

## 二、短视频引流脚本

### 视频1：痛点共鸣型（30秒，竖屏9:16）

**目标**：让银行人看到自己 → 点击主页 → 进落地页

**脚本**：

>

**制作提示**：
